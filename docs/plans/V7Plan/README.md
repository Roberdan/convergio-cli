## üìù Key Decisions (Final)

### Architecture
- **Hybrid Approach:** Core C (performance-critical) + Rust (API Gateway, plugins)
- **LLM Strategy:** MLX (Apple Silicon) + Ollama (cross-platform) + Cloud (quality)
- SvelteKit web UI (primary)
- PostgreSQL/Redis for state
- Cross-platform: macOS, Web, Linux, Windows (future)

### Pricing
- Free: 30 questions/month
- Pro: $6.99/month + 100 questions + $0.01/extra
- Team: $19.99/month + 500 questions + $0.01/extra

### Timeline
- **Sequential:** 12 months (baseline)
- **Parallel:** 6-8 months (optimized, 33-50% faster)
- Months 1-2: Foundation (all teams parallel)
- Months 3-4: Integration (connect components)
- Months 5-6: Enhancement (polish & scale)
- Months 7-8: Launch (beta & public)

**See [V7Plan-PARALLEL-DEVELOPMENT.md](./V7Plan-PARALLEL-DEVELOPMENT.md) for parallel strategy.**
